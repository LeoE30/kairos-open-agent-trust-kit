# Grant Scope

This repository is suitable for public-good grants because it produces reusable trust primitives rather than a closed commercial system.

## What A Grant Would Fund

- open schemas;
- local-first demo implementation;
- documentation and examples;
- ecosystem adapter prototypes;
- short technical report;
- public demo video;
- founder and engineering time required to deliver the above.

## What A Grant Would Not Fund

- closed Forneus commercial infrastructure;
- private customer deployments;
- confidential internal architecture;
- military operational workflows;
- token issuance or speculative economics.

## Why It Matters

Agentic AI needs a trust layer. Builders should be able to add human approval and evidence receipts before deploying agents into sensitive workflows. This project gives them a simple starting point.
