# Roadmap

## Phase 1 - Public Schemas

- Stabilize task intake, risk classification, and evidence receipt schemas.
- Add JSON examples for common AI agent workflow patterns.
- Define minimal privacy rules for what should never be exported.

## Phase 2 - Local Reference Demo

- Expand the deterministic demo policy.
- Add optional local signature support.
- Add export examples for compliance, research, and security review workflows.

## Phase 3 - Ecosystem Adapters

- Publish adapter examples for common agent orchestration frameworks.
- Keep adapters small, inspectable, and non-invasive.
- Avoid dependency on a single cloud provider or model API.

## Phase 4 - Review And Documentation

- Publish a short technical report.
- Collect feedback from open-source AI, decentralized compute, and public-good infrastructure communities.
- Prepare a final grant milestone report with demo video and reproducible examples.
